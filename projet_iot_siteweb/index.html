Système de Monitoring IoT Local : ESP32, Raspberry Pi et Interface Web Sécurisée

1. Contexte et objectifs

Ce projet consiste en la création d'un système de monitoring IoT local utilisant un ESP32 pour la collecte de données, un Raspberry Pi pour le traitement et l'affichage des données, et une interface web sécurisée pour la visualisation des informations.

Le système permet de surveiller divers paramètres environnementaux tels que la température, l'humidité, et d'autres données pertinentes, tout en garantissant la sécurité des données via une interface web protégée.

2. Architecture Matérielle et Logicielle

Le système est organisé autour des éléments suivants :

  • Un ESP32 servant de nœud IoT pour capter les données (température et humidité).
  • Une Raspberry Pi connectée en Ethernet à un routeur local, jouant le rôle de serveur.
  • Un serveur LAMP (Linux, Apache, MySQL, PHP) installé sur la Raspberry Pi.
  • Une base de données MySQL gérée via l’interface PhpMyAdmin.
  • Un site web dynamique relié à la BDD qui récupère les données.
  • Un routeur WiFi assurant la liaison réseau locale entre les composants.

3. Fonctionnement du Site Web

Le site web comporte plusieurs pages permettant à l’utilisateur de :

  • Accéder à une page d’accueil présentant le projet.
  • Se connecter via un formulaire sécurisé (email + mot de passe).
  • Accéder à un tableau de bord avec les mesures du capteur.

Deux versions du site sont proposées :

  • Une version de développement autonome avec données insérées manuellement dans la base de données.
  • Une version connectée à l’ESP32 qui envoie des données en temps réel via HTTP POST vers un script PHP.
